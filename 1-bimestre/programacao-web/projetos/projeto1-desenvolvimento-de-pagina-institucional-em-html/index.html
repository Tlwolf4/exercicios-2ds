Conecta jovem

A ONG Conecta Jovem é uma organização que oferece cursos gratuitos de tecnologia voltados para adolescentes, com o objetivo de promover inclusão digital e ampliar oportunidades de aprendizado. Por meio de aulas práticas e acessíveis, a instituição ensina habilidades como informática básica, uso seguro da internet e noções de programação, ajudando os jovens a se prepararem melhor para os estudos e para o futuro profissional. Dessa forma, a ONG contribui para o desenvolvimento educacional e social dos participantes.

Cursos Oferecidos

  • CSS
  • JavaScript
